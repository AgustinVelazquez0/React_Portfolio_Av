# AV — Brand Kit

Personal engineering brand of **Agustin Velazquez**.

> *"Code is the medium. Detail is the difference."*

---

## 1. Brand Essence

| Attribute | Value |
|---|---|
| Name | AV — Agustin Velazquez |
| Tagline | Full-stack craft. AI-native mindset. |
| Mission | Ship precise, scalable products where code, AI and design converge. |
| Mood | Precise · Intentional · Timeless |

## 2. Logo Suite

| File | Use |
|---|---|
| `logo-monogram.svg` | Primary mark. Uses `currentColor` — inherits text color. Use on light/dark backgrounds. |
| `logo-horizontal.svg` | Logo + name + role. Use in headers, signatures, footers. |
| `logo-icon.svg` | App icon, badges, social avatars. Cyan square + black mark. |
| `/favicon.svg` | Browser tab icon (same as logo-icon). |

### Clear space
Reserve a margin of **at least 1×** the height of one peak around the mark.

### Don'ts
- Don't change the stroke weight ratio.
- Don't add gradients, shadows or 3D effects.
- Don't rotate or skew.
- Don't recolor the cyan to other accent colors.

## 3. Color Palette

| Token | Hex | Use |
|---|---|---|
| Neon Cyan | `#22D3EE` | Primary brand accent, CTAs, links, highlights |
| Void | `#0A0A0A` | Primary dark surface |
| Charcoal | `#1A1A1A` | Secondary dark surface, cards |
| Steel | `#404040` | Borders, dividers, muted text on dark |
| Amber | `#F59E0B` | Reserved for high-priority alerts and "available" status |
| Pure | `#FFFFFF` | Primary light surface, body text on dark |

## 4. Typography

| Family | Use | Weights |
|---|---|---|
| **Geist Sans** | UI, body, headings | 400 / 500 / 600 / 700 |
| **JetBrains Mono** | Code, numerics, micro-labels (uppercase tracked) | 400 / 500 / 700 |

Fallback stack: `system-ui, -apple-system, sans-serif`.

## 5. Voice & Tone

- **Direct.** No marketing fluff.
- **Technical.** Use precise terminology (stack names, metrics, architectures).
- **Confident, not arrogant.** State outcomes; let the work speak.
- **Bilingual.** Spanish primary, English-ready (i18n in product).

## 6. Application

- **Web:** dark mode default; cyan as the only chromatic accent.
- **Social:** square logo on dark backgrounds; never on busy photos.
- **Documents (CV / proposals):** monogram top-left; name in Geist Sans 600; metadata in JetBrains Mono uppercase.
